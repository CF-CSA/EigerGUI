This directory provides three files:
- process_eiger.sh
  A bash script defining functions to process data with xds and scale with
  sadabs. Please take a look inside for required binaries
- XDS-D8-Eiger2R500.INP
  template for XDS.INP
- XDS-D8-Eiger2R500_OFFSETS.INP
  instrument specific offsets can be applied here. The syntax is similar to XDS
